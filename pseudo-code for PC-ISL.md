
## Detailed Pseudocode and Annotations for Complete Training Process

This document provides the complete pseudocode for the PC-ISL method.

---

## Algorithm Overview

---

## Main Symbol Definitions

- `X_train`: Original training data
- `X_aug`: Augmented contrastive learning data
- `X_test`: Test dataset
- `X_valid`: Valid dataset
- `y_true`: True labels
- `y_pseudo`: Pseudo labels (generated by clustering)
- `E`: Encoder
- `P`: Projection Head
- `C`: Classifier
- `K`: Number of clusters
- `N_classes`: Number of true classes
- `samples_per_class`: Number of samples per class

---

## Complete Pseudocode

```
Algorithm: PC-ISL with 10-Shot Valid Sampling

Input:
    X_train: Original training dataset
    X_aug: Augmented contrastive learning dataset
    X_test: Test dataset with true labels
    X_valid: Validation data with true labels
    K: Number of clusters
    N_classes: Number of true classes
    samples_per_class: Number of samples per class
    N_outer: Maximum number of outer loop iterations
    seed: Random seed

Output:
    E*: Optimal encoder
    C*: Optimal classifier
    best_acc_full: Best accuracy on full test set

    
-------------------------------------------------------------------
Phase 1: Initialization and Data Preparation
-------------------------------------------------------------------

1. [Environment Initialization]
   Set random seed (seed=42)                    // Ensure reproducibility
   Initialize GPU device                        // Use CUDA for acceleration
   Create logger                                // Record training process and results
   Log: "Test data sampling experiment - {samples_per_class} samples per class"

2. [Data Loading]
   Load original training data X_train          // Unaugmented original WiFi CSI data
   Load augmented contrastive data X_aug        // Data-augmented contrastive learning data
   Load test data X_test, y_test                // Test set for final evaluation
   Load validation data X_valid, y_valid        // Validation set for early stopping

3. [Model Initialization]
   Initialize encoder E(input_shape, z_dim=128)     // Extract feature representations
   Initialize projection head P(z_dim=128, output=128)  // For contrastive learning
   Build SimCLR model = E + P                       // Combine encoder and projection head
   Load pretrained weights from Stage1              // Load model trained in first stage
   Initialize classifier C(z_dim=128, num_classes=K)  // K-way classifier
   
4. [Early Stopping Mechanism Initialization]
   best_acc_outer ← 0.0                         // Record best accuracy in outer loop (validation set)
   patience_counter ← 0                         // Early stopping counter
   best_iteration ← 0                           // Record best iteration round


-------------------------------------------------------------------
Phase 2: Outer Iteration Loop
-------------------------------------------------------------------

For outer_iter = 1 to N_outer:

    5. [Iteration Start Recording]
       Record current iteration start time
       Log: "===== Outer Loop Round outer_iter ====="
       
    6. [Model State Recovery]
       If outer_iter > 1:
           Load best encoder E* from previous round  // Continue from previous optimal state
       Reinitialize classifier C                     // Retrain classifier each round
       
    -------------------------------------------------------------------
    Step A: Clustering & Pseudo-labeling
    -------------------------------------------------------------------
    
    7. [Feature Extraction]
       features_all ← []
       labels_all ← []
       
       For batch in X_train:
           z ← E(batch)                          // Extract features using encoder
           features_all.append(z)
           labels_all.append(batch.labels)
       
    8. [K-means Clustering]
       kmeans ← KMeans(n_clusters=K)
       cluster_labels ← kmeans.fit_predict(features_all)
       Compute NMI(cluster_labels, labels_all)   // Evaluate clustering quality
       
    9. [Pseudo Label Mapping]
        For each augmented sample x_aug in X_aug:
            Find corresponding original sample index idx
            y_pseudo[x_aug] ← cluster_labels[idx]
       
    10. [Positive Pair Mask Generation]
        positive_mask ← Create positive pair mask
        For i, j in sample pairs:
            If y_pseudo[i] == y_pseudo[j]:
                positive_mask[i,j] ← 1
            Else:
                positive_mask[i,j] ← 0
    
    11. [Save Clustering Results]
        Save aug_data, aug_pseudo_labels, aug_original_labels
        To directory: cluster_result/outer_iter_{outer_iter}/
    
    -------------------------------------------------------------------
    Step B: Confidence-based Sample Selection
    -------------------------------------------------------------------
    
    12. [Classifier Training Preparation]
        optimizer_C ← Adam(C.parameters(), lr=0.001)
        criterion ← CrossEntropyLoss()
        
        Initialize sample tracking variables:
        sample_loss_history ← {}
        sample_pred_history ← {}
        convergence_speed ← {}
        
    13. [Classifier Iterative Training]
        For epoch = 1 to max_epochs (1000):
        
            For batch in DataLoader(X_aug, y_pseudo):
            
                a) [Forward Propagation]
                   z ← E(batch.data)                  // Encoder extracts features (frozen)
                   logits ← C(z)                      // Classifier prediction
                   loss ← criterion(logits, batch.pseudo_labels)
                
                b) [Sample-level Loss Recording]
                   For each sample i in batch:
                       sample_loss_history[i].append(loss[i])
                       pred_i ← argmax(logits[i])
                       sample_pred_history[i].append(pred_i)
                
                c) [Backward Propagation]
                   optimizer_C.zero_grad()
                   loss.backward()
                   optimizer_C.step()
            
            d) [Compute Training Metrics]
               train_acc ← 100 × correct / total        // Training accuracy
               train_loss ← running_loss / len(aug_loader)  // Average training loss
               
               Record epoch_losses[epoch] ← train_loss
               Record epoch_accuracies[epoch] ← train_acc
               
               Log: "Epoch {epoch}: Loss: {train_loss:.4f}, Acc: {train_acc:.2f}%"
            
            e) [Early Stopping Check] (based on training loss)
               If train_loss < best_loss - min_delta:
                   best_loss ← train_loss               // Update best loss
                   patience_counter ← 0                 // Reset counter
               Else:
                   patience_counter += 1                // Increment counter
                   Log: "EarlyStopping counter: {patience_counter} out of {patience}"
               
               If patience_counter >= patience:
                   Log: "Early stopping triggered, stop training"
                   Break loop                           // Stop classifier training
        
        // After training, save final classifier model
        Save classifier state C_final
        Log: "Classifier training completed, final model saved"
    
    14. [Confidence Scoring]
        
        a) [Loss Convergence Score]
           // Evaluate sample reliability based on loss convergence speed
           For each sample i in sample_loss_history:
               loss_trajectory ← sample_loss_history[i]
               final_loss ← loss_trajectory[-1]
               
               // Find epoch where loss converges (loss <= 1.1 × final_loss)
               e_conv ← Find first epoch where loss_trajectory[e] <= 1.1 × final_loss
               If e_conv not found:
                   e_conv ← len(loss_trajectory) - 1
               
               // Faster convergence, higher score
               s_loss_conv[i] ← 1.0 / (e_conv + 1)
        
        b) [Cross-view Consistency Score]
           // Evaluate prediction stability across different augmented views
           For each sample i:
               paired_indices ← Find all indices j where positive_mask[i,j] == 1
               
               If len(paired_indices) > 0:
                   j ← paired_indices[0]
                   With torch.no_grad():
                       pred_i ← argmax(C(E(aug_data[i])))
                       pred_j ← argmax(C(E(aug_data[j])))
                   s_consist[i] ← 1 if pred_i == pred_j else 0
               Else:
                   s_consist[i] ← 0
        
        c) [Comprehensive Confidence Score]
           // Adaptive weights: loss convergence weight increases with iterations
           t ← outer_iter + 1
           ω_1 ← 0.05 + 0.5 × (1 - exp(-0.05 × (t - 1)))  // Loss convergence weight
           ω_2 ← 1 - ω_1                                   // Cross-view consistency weight
           
           For each sample i:
               comprehensive_scores[i] ← ω_1 × s_loss_conv[i] + ω_2 × s_consist[i]

    15. [Trusted Sample Selection]
        Sort all samples by Confidence in descending order
        Select top-p% samples as trusted sample set D_trusted
        Create confidence mask confidence_mask:
            confidence_mask[i] ← 1 if i in D_trusted  // Trusted sample
            confidence_mask[i] ← 0 otherwise          // Untrusted sample
        
        Log: "Selected {len(D_trusted)} trusted samples"

    
    -------------------------------------------------------------------
    Step C: Joint Training of Encoder and Classifier
    -------------------------------------------------------------------
    
    16. [Joint Training Preparation]
        Reinitialize classifier C
        optimizer_joint ← AdamW(E.parameters() + C.parameters(), lr=0.00005, weight_decay=0.05)
        scheduler ← CosineAnnealingWarmRestarts(T_0=10, T_mult=2, eta_min=1e-6)
        criterion_CE ← CrossEntropyLoss()
        criterion_CL ← NTXentLoss(temperature=0.07)
        
        // Early stopping mechanism initialization
        best_valid_acc ← 0.0
        patience_counter ← 0
        patience ← 5
        min_delta ← 0.001
        
    17. [Joint Training Iteration]
        For epoch = 1 to max_epochs (300):
        
            E.train()
            C.train()
            total_loss ← 0.0
            correct ← 0
            total ← 0
            
            For batch in DataLoader(D_trusted, batch_size=32):
                // Unpack batch data
                anchor, positive, anchor_label, positive_label, batch_confidence_mask ← batch
                
                // 1. Supervised training (trusted samples)
                If batch_confidence_mask.any():
                    trusted_inputs ← anchor[batch_confidence_mask]
                    trusted_targets ← anchor_label[batch_confidence_mask]
                    
                    If len(trusted_inputs) > 0:
                        features ← E(trusted_inputs)
                        logits ← C(features)
                        loss_supervised ← criterion_CE(logits, trusted_targets)
                        
                        // Compute accuracy
                        predicted ← argmax(logits)
                        correct += sum(predicted == trusted_targets)
                        total += len(trusted_targets)
                    Else:
                        loss_supervised ← 0.0
                Else:
                    loss_supervised ← 0.0
                
                // 2. Unsupervised training (untrusted samples)
                If (~batch_confidence_mask).any():
                    untrusted_indices ← where(~batch_confidence_mask)
                    untrusted_anchors ← anchor[untrusted_indices]
                    untrusted_positives ← positive[untrusted_indices]
                    
                    If len(untrusted_anchors) > 0:
                        anchor_features ← E(untrusted_anchors)
                        positive_features ← E(untrusted_positives)
                        loss_unsupervised ← criterion_CL(anchor_features, positive_features)
                    Else:
                        loss_unsupervised ← 0.0
                Else:
                    loss_unsupervised ← 0.0
                
                // 3. Compute total loss (weighted combination)
                supervised_weight ← 1.0
                unsupervised_weight ← 0.8
                loss ← supervised_weight × loss_supervised + unsupervised_weight × loss_unsupervised
                
                // 4. Backward propagation and optimization
                optimizer_joint.zero_grad()
                loss.backward()
                clip_grad_norm(E.parameters(), max_norm=1.0)
                clip_grad_norm(C.parameters(), max_norm=1.0)
                optimizer_joint.step()
                
                total_loss += loss.item()
            
            scheduler.step()
            
            // Compute training metrics
            avg_loss ← total_loss / len(DataLoader)
            train_acc ← 100 × correct / total
            
            // Evaluate validation set
            valid_acc ← evaluate_classifier(E, C, X_valid, y_valid)
            
            Log: "Epoch {epoch}: Loss={avg_loss:.4f}, Train Acc={train_acc:.2f}%, Valid Acc={valid_acc:.2f}%"
            
            // Early stopping check
            If valid_acc > best_valid_acc + min_delta:
                best_valid_acc ← valid_acc
                patience_counter ← 0
                Save best model E*, C*
            Else:
                patience_counter += 1
                If patience_counter >= patience:
                    Log: "Early stopping triggered, stop joint training"
                    Break loop
    
    18. [Outer Loop Evaluation]
        // Evaluate validation set using best model
        Load best model E*, C*
        current_acc ← evaluate_classifier(E*, C*, X_valid, y_valid)
        
        Log: "Outer loop {outer_iter} completed, validation accuracy: {current_acc:.2f}%"
        
        // Outer loop early stopping check
        If current_acc > best_acc_outer + min_delta:
            best_acc_outer ← current_acc
            best_iteration ← outer_iter
            patience_counter_outer ← 0
            Save best outer model E_best, C_best
        Else:
            patience_counter_outer += 1
            If patience_counter_outer >= patience_outer:
                Log: "Outer loop early stopping triggered"
                Break outer loop

-------------------------------------------------------------------
Phase 3: Final Evaluation
-------------------------------------------------------------------

19. [Final Test Evaluation]
    Load best model E_best, C_best
    final_acc ← evaluate_classifier(E_best, C_best, X_test, y_test)
    
    Log: "Final test set accuracy: {final_acc:.2f}%"
    Log: "Best iteration round: {best_iteration}"
    
    Return: E_best, C_best, final_acc

Algorithm End
```

## Key Features of PC-ISL

### 1. Progressive Confidence-based Learning
- Iteratively refines sample confidence through multiple outer loops
- Combines loss convergence speed and cross-view consistency for confidence scoring
- Adaptive weighting mechanism that evolves with training progress

### 2. Hybrid Training Strategy
- Supervised learning on high-confidence samples with pseudo labels
- Unsupervised contrastive learning on low-confidence samples
- Balanced combination prevents overfitting to noisy pseudo labels

### 3. Multi-level Early Stopping
- Classifier training: Based on training loss
- Joint training: Based on validation accuracy
- Outer loop: Based on validation accuracy across iterations
- Prevents overfitting and reduces computational cost

### 4. Validation-based Model Selection
- Uses small validation set (10-shot) for early stopping during training
- Final evaluation on complete test set
- Ensures fair comparison and prevents test set leakage

## Algorithm Complexity

### Time Complexity
- Feature extraction: O(N × d)
- K-means clustering: O(N × K × d × I_kmeans)
- Classifier training: O(E_cls × N × d)
- Joint training: O(E_joint × N × d)
- Total per outer iteration: O(N × d × (K × I_kmeans + E_cls + E_joint))

Where:
- N: Number of training samples
- d: Feature dimension
- K: Number of clusters
- I_kmeans: K-means iterations
- E_cls: Classifier training epochs
- E_joint: Joint training epochs

### Space Complexity
- Model parameters: O(d²)
- Sample history: O(N × E_cls)
- Confidence scores: O(N)
- Total: O(d² + N × E_cls)

---

## Hyperparameter Guidelines

### Critical Hyperparameters
- `K` (clusters): Should match or slightly exceed true number of classes
- `patience`: 5 epochs for inner loops, 3 iterations for outer loop

### Learning Rates
- Classifier training: 0.001
- Joint training: 0.00005 with weight decay 0.05
- Scheduler: CosineAnnealingWarmRestarts (T_0=10, T_mult=2)

### Loss Weights
- Supervised weight: 1.0 (fixed)
- Unsupervised weight: 0.8 (can be adjusted based on data quality)
